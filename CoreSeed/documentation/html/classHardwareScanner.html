◆ getCPUModel()

QString HardwareScanner::getCPUModel ( )

Ermittelt das CPU-Modell via sysctl.

WAS: CPU-Modell auslesen.

Rückgabe
String mit der CPU-Bezeichnung.

WIE: Aufruf von 'sysctl -n hw.model'. WARUM: Um dem Benutzer anzuzeigen, auf welcher Hardware er installiert.

◆ getDisks()

QStringList HardwareScanner::getDisks ( )

Listet die im System gefundenen Festplatten auf.

WAS: Festplattennamen ermitteln.

Rückgabe
Liste der Gerätenamen (z.B. sd0, wd0).

WIE: Abfrage von 'sysctl -n hw.disknames'. WARUM: Identifikation der Zielmedien für die Installation.
